News Credibility Checker
What it does, what problems it solves, and how it works

What is TruthLens?
TruthLens is an AI-powered news credibility checker. It is a single HTML file — no installation, no login, no payment required. Just open it in any browser and it works. It uses the Claude AI engine to analyse news articles and give you a credibility score from 0 to 100, along with a full explanation of why that score was given.

How to Use It

Open the file in your browser
Paste the article URL and copy-paste the full article text
Click "Initiate Credibility Check"
Watch the live verification process run in real time
Read your full credibility report


What Problems It Solves
❌ Problem 1 — Fake news spreads faster than corrections and most people have no quick way to verify a story
✅ Solution — TruthLens gives a full verification report in under 10 seconds
❌ Problem 2 — Professional fact-checking tools are expensive and require technical knowledge
✅ Solution — One HTML file, works in any browser, completely free to use
❌ Problem 3 — AI tools give verdicts without explaining why, so users can't trust them
✅ Solution — Every single scoring decision is logged and explained. Nothing is hidden.
❌ Problem 4 — Readers can't recognise clickbait and sensationalist language patterns
✅ Solution — Scans 21 clickbait trigger words and highlights them directly in the article text in red
❌ Problem 5 — A story on only one website is less reliable but readers don't know to check
✅ Solution — Automatically cross-checks the article against 10 major news sources and adjusts the score based on how many trusted outlets covered the same story

Features
🤖 AI Analysis — Claude AI reads the article first, extracts keywords, detects sensationalism, and writes a plain-English summary before any scoring begins
📊 Credibility Score (0–100) — Starts at 50, five modules add or subtract points, final score clamped between 0 and 100
🔗 Multi-Source Verification — Checks against BBC News, Reuters, AP News, New York Times, The Guardian, The Hindu, Indian Express, NDTV, and 2 untrusted sources for comparison
✅ Matched Sources Only — The results page shows ONLY the websites where the article was found. No clutter. If nothing matched, it clearly says so.
🔍 Text Analysis — Detects ALL CAPS abuse, excessive punctuation, clickbait phrases, vague attribution like "experts say", and positive fact indicators like named sources and statistics
📋 Full Transparency Report — 4 expandable panels showing: how the score was calculated, why sources matched or didn't, flagged words highlighted in the article, and a full step-by-step decision log
⚡ 3-Screen Flow — Input screen → Live animated verification pipeline → Full results report. Each screen completes before the next appears. You watch every step happen live.

Important Honest Disclosures
⚠️ TruthLens does not browse the live internet. Browser security (CORS) prevents direct requests to external sites. Source matching is done through AI keyword analysis — it simulates what a real multi-source search would find based on the article's content.
⚠️ A high score does not guarantee the article is true. A low score does not guarantee it is false. This is a decision-support tool — human judgment is still required.
⚠️ Works best with English-language articles. Other languages will still get a score based on domain and text quality but language-specific analysis won't apply.
